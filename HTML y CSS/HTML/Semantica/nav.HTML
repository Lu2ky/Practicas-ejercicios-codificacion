Uso de la sentencia nav

Para hacer uso de la sentencia nav debemos tener claro
que es lo que queremos poner alli, nav basicamente es
la barra de seleccion de paginas la cual nos ayuda a movernos
entre distintas paginas. 


    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
    </head>
    <body>
    <nav>
        <ul>
            <li><a href = "test.HTML">Esto</a></li>
            <li><a href = "test.HTML">es</a></li>
            <li><a href = "test.HTML">un</a></li>
            <li><a href = "test.HTML">nav</a></li>
        </ul>
    </nav>
    </body>
    </html>
