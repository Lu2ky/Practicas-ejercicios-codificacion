Uso de la sentencia main

El elemento <main> en HTML se utiliza para definir el contenido principal y único de un documento o aplicación web.
Dentro de él se coloca la información más relevante de la página, aquella que está directamente relacionada con el propósito del documento.
No debe incluir elementos que se repitan en todas las páginas, como menús de navegación, logotipos o información de pie de página, ya que esos corresponden a <header> o <footer>.
A diferencia de <header> o <footer>, solo puede existir un único <main> por documento, lo que ayuda a los navegadores y tecnologías de asistencia a identificar de forma clara el núcleo de la página.

Ejemplo: 


    <!DOCTYPE html>
    <html lang="en">
    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>Document</title>
    </head>
    <body>
      <main>
        Esto es el contenido principal de la página
      </main>
    </body>
    </html>
