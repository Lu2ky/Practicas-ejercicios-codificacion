Uso de la sentencia header

El elemento <header> en HTML se utiliza para definir la parte introductoria o de cabecera de una página o de una sección específica.
Dentro de él se suelen colocar elementos como el título principal, logotipos, menús de navegación o cualquier información que sirva de presentación.
No debe confundirse con <head>, que pertenece a la configuración del documento.
Además, un mismo documento puede tener varios <header>, por ejemplo, uno general en la página y otros en artículos o secciones individuales.

Ejemplo: 


    <!DOCTYPE html>
    <html lang="en">
    <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    </head>
        <body>
    <header>
        Esto es un header
    </header>
    </body>
    </html>
