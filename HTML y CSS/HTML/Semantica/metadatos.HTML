Metadatos

Los metadatos son utilizados para darle facilidades a los motores de busqueda
estos mismos analizan el documento HTML y contenido multimedia para determinar la relevancia

Las meta etiquetas definen metadatos sobre una pagina web
se añaden dentro del elemento <head> osea no es algo visible
Estructura basica: 

  <meta name="nombre" content="valor">
estos metadatos proporcionan informacion valiosa a los motores de busqueda

Metaetiquetas basicas para SEO(Search Engine Optimization)

  • <meta name="description" content=""> = Se añade una descripcion breve de la pagina
  • <meta name="title" content=""> = Se pone el titulo de la pagina
  • <meta name="autor" content=""> = Se pone el nombre del autor
  • <meta name="language" content=""> = Especifica el idioma de la pagina
  • <meta name="robots" content=""> = Indica a los buscadores como rastrear la pagina
      Content:
    • "index" = analiza la pagina
    • "follow" = visita los enlaces
    • "noindex" = no analiza la pagina
    • "nofollow" = no visita los enlaces
  • <meta name="google" content=""> = Controla como Google mustra los resultados de busqueda
      Content:
    • "noindex" = evita que google indexe la pagina
    • "nosnippet" = no muestra fragmentos de texto ni cache en los resultados
    • "noarchive" = evita que google indexe la pagina
    • "notranslate" = evita que Google ofrezca traducir la pagina
    • "noimageindex" = evita que las imagenes de la pagina se indexen en Google imagenes
    • "unavailable_after: [fecha]" = indica cuando dejar de mostrar la pagina en resultados
      Ejemplo: unavailable_after: 25-Sep-2025 15:00:00 EST
    • "nositelinkssearchbox" = evita que aparezca el cuadro de busqueda de sitio en los resultados
  • <meta name="googlebot" content=""> = Controla como Google muestra los resultados de busqueda, tiene el content de robots y de google
  • <meta name="revised" content="date"> = Especifica la fecha de ultima modificacion
  • <meta name="rating" content=""> = Especifica la audiencia prevista
      • Content:
    • general
    • 14 years
    • safe for kids
    • mature
  • <meta name="copyright" content="Copyright 2022"> = Especifica informacion de copyright

Etiquetas meta http-equiv

  • <meta http-equiv="content-type" content="text/html"> = Formato del documento
  • <meta http-equiv="default-style" content=""> = Formato del documento de estilo (CSS)
  • <meta http-equiv="refresh" content=""> = Duracion antes de considerar la pagina obsolote
  • <meta http-equiv="Content-language" content=""> = Especifica el idioma
  • <meta http-equiv="Cache-Control" content="no-cache"> = Controla el almacenamiento en cache

Metaetiquetas para diseño responsive

  • <meta name="format-detection" content="telephone=yes"> = Hace numeros telefonicos clicables
  • <meta name="HandheldFriendly" content="true"> = Indica compatibilidad con dispositivos moviles
  • <meta name="viewport" content="width=device-width initial-scale=1.0"> = Configura el area visible en dispositivos moviles
