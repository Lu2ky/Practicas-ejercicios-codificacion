Inputs en HTML

Los inputs son herramientas que nos ayudan para hacer una correcta recoleccion de datos del usuarios
estos son variados, menus, checkbox,passwords, text, etc.

  • <button "action" = "output"> = Crea un boton el cual tiene multiples acciones y salidas
    • Actions:
    • onclick = Se activa cuando haces un clic simple sobre el botón.
    • ondblclick = Se activa cuando haces doble clic rápido sobre el botón.
    • onmousedown = Se activa cuando presionas un botón del mouse encima del botón (antes de soltarlo).
    • onmouseup = Se activa cuando sueltas el botón del mouse encima del botón.
    • onmouseover = Se activa cuando el puntero entra en el área del botón.
    • onmouseout = Se activa cuando el puntero sale del área del botón.
    • onmousemove = Se activa cada vez que mueves el puntero dentro del botón.
    • oncontextmenu = Se activa cuando haces clic derecho sobre el botón (menú contextual).
    • onkeydown = Se activa cuando presionas una tecla del teclado mientras el botón tiene el foco.
    • onkeyup = Se activa cuando sueltas una tecla mientras el botón tiene el foco.
    • onkeypress = Se activa mientras mantienes una tecla presionada (deprecated, mejor usar onkeydown/onkeyup).
    • onfocus = Se activa cuando el botón recibe el foco (ej: con Tab o clic).
    • onblur = Se activa cuando el botón pierde el foco.
    • onsubmit = Se activa cuando el botón es de tipo "submit" y envía un formulario.
    • onreset = Se activa cuando el botón es de tipo "reset" y reinicia un formulario.
    • ondragstart = Se activa cuando empiezas a arrastrar el botón.
    • ondrop = Se activa cuando sueltas (drop) algo arrastrado encima del botón.
    • onwheel = Se activa cuando usas la rueda del mouse estando sobre el botón.
  • <input type = "checkbox"> = Crea un boton de chequeo
    • Ejemplo: 
    
                <input type="checkbox" id="dog" name="dog" value="Dog">
                <label for="dog">I like dogs</label>
                <input type="checkbox" id="cat" name="cat" value="Cat">
                <label for="cat">I like cats</label>
  • <input type="radio"> = Muestra multiples botones los cuales permiten seleccionar un unico valor entre varios.
    • Ejemplo: 
     
                    <input type="radio" id="light" name="theme" value="Light"> 
                    <label for="light">Light</label> 
                    <input type="radio" id="dark" name="theme" value="Dark"> 
                    <label for="dark">Dark</label>
  • <input type="time"> = Campo para seleccionar hora.
    • Ejemplo: 
    
        <input type="time" id="hora" name="hora">
  • <input type="color"> = Selector de color.
    • Ejemplo: 
    
        <input type="color" id="favcolor" name="favcolor" value="#ff0000">
  • <input type="tel"> = Campo para números de teléfono (puede validar un patrón).
    • Ejemplo: 
    
        <input type="tel" id="phone" name="phone" placeholder="123-456-7890" pattern="[0-9]{3}-[0-9]{3}-[0-9]{4}">
  • <input type="search"> = Campo optimizado para búsqueda.
    • Ejemplo: 
    
        <input type="search" id="busqueda" name="busqueda" placeholder="Buscar...">
  • <input type="file"> = Permite al usuario subir archivos.
    • Ejemplo: 
    
        <input type="file" id="archivo" name="archivo">
  • <input type="submit"> = Botón que envía el formulario.
    • Ejemplo: 
    
        <input type="submit" value="Enviar">
  • <input type="reset"> = Botón que reinicia los campos del formulario.
    • Ejemplo: 
    
        <input type="reset" value="Reiniciar">
  • <input type="hidden"> = Campo oculto que no se muestra al usuario pero envía datos.
    • Ejemplo: 
    
        <input type="hidden" id="userid" name="userid" value="12345">
  • <input type="datetime-local"> = Campo para fecha y hora local combinadas.
    • Ejemplo: 
    
        <input type="datetime-local" id="fechaHora" name="fechaHora">
  • <input type="text"> = Campo de texto donde el usuario puede escribir libremente.
    • Ejemplo: 
    
        <input type="text" id="nombre" name="nombre" placeholder="Escribe tu nombre">
