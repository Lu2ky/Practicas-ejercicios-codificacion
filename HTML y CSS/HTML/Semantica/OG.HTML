Open Graph Protocol

Los metadatos Open Graph Protocol son utilizados para hacer vistas previas
al compartir el enlace de la pagina, donde se puede modificar titulo,tipo,url de la pagina principal,imagen de vista previa, etc.
Estructura basica: 

 <meta property="og:{Content}" content="valor">

Metadatos OG:

  • <meta property="og:title" content=""> = Especifica el titulo de la pagina
  • <meta property="og:type" content=""> = Especifica el tipo de contenido (web,imagen,musica,video,etc)
  • <meta property="og:url" content=""> = Url de la web
  • <meta property="og:image" content=""> = Imagen de la vista previa
  • <meta property="og:description" content=""> = Indica la descripcion
  • <meta property="og:locale" content=""> = Indica el idioma y la configuracion regional de la pagina
  • <meta property="og:site_name" content=""> = Indica el nombre del sitio
