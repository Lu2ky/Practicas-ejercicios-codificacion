Uso de la sentencia footer

El elemento <footer> en HTML se utiliza para definir el pie de página de un documento o de una sección específica.
Dentro de él se suele colocar información de contacto, derechos de autor, enlaces relacionados o notas finales.
No debe confundirse con <header>, que representa la parte introductoria o cabecera.
A diferencia de <main>, pueden existir varios <footer> en un documento, por ejemplo, uno general en la página y otros en artículos o secciones individuales.

Ejemplo: 


    <!DOCTYPE html>
    <html lang="en">
    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>Document</title>
    </head>
    <body>
      <footer>
        Esto es un pie de página
      </footer>
    </body>
    </html>
